# 简易文件内容统计工具

### 功能说明：

输入指定文件名，可统计该文本文件的基本信息（如行数、单词数），生成简单可视化图表

### 编译步骤

##### 环境要求：

c++编译环境

windows

##### 编译命令

```cmake
mkdir build_file
cd build_file
cmake ..
cmake --build .
```

### 实现过程中遇到的问题及解决方法：

1.在实现判断空行功能时，一开始直接利用empty判断该字符串是否为空，但它只能识别空字符串，如果行里面含有空格并不会被识别为空行。

解决方法：使用if判断语句，考虑空行的所有可能性

2.在处理单词时，没有考虑最后一个单词（其后没有标点符号）

解决方法：增加判断模块
